# flip

## Required

that is required to be installed
- React-native-cli
- NPM (Node Package Manager)
- Emulator/Simulator

## Clone or download
- `$ git clone https://github.com/auliarampit/flip.git`
- `$ cd flip`
- `$ npm install`
- `$ pod install`

## Start Application
  `$ yarn android`
  `$ yarn ios`
