# flip

## Required

that is required to be installed
- React-native-cli
- NPM (Node Package Manager)
- Emulator/Simulator

## Clone or download
- `$ Apk Demo https://drive.google.com/file/d/16hcp17P7qXXgt14fJiUJI9LUMfMGVXx8/view?usp=sharing`
- `$ git clone https://github.com/auliarampit/flip.git`
- `$ cd flip`
- `$ npm install`
- `$ pod install`

### Start Application
  `$ yarn android`
  `$ yarn ios`
